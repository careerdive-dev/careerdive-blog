## 커리어다이브가 전하는 생생한 이야기


**몰입하는 사람들**
- 게임 유저에서 게임 기획자로 (게임 기획자, 컴투스편)
- 트렌드에 민감한 내가 PD가 되었을 때 (방송·뉴미디어 PD, 재능방송편)
- 내가 믿는 나의 감각, 그리고 회사가 그걸 밀어준다? (대외홍보, 롯데이노베이트편)
- 좋아하는 브랜드의 마케터로 일한다는 건 (마케터, 비밀😉)
- 기술의 최전선에 서겠다는 꿈을 이루는 중 (반도체 설계 엔지니어, SK 하이닉스)
- 내가 디자인한 작업물을 일상에서 마주할 때 (시각디자이너, 비밀😉)
- 프랑스어를 사용하는 순수한 즐거움 (기내 통역원, 에어프랑스)
